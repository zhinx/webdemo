# Netease Java Web Project

* 开发环境

  * IntelliJ IDEA 2017

* 项目构建

  * Maven
  * Spring MVC + MyBatis
  * MySQL 5.7
  * Tomcat 8

* 数据库配置

  * 配置脚本：schema.sql

  ```
  jdbc.url = jdbc:mysql://localhost:3306/webdemo?useUnicode=true&characterEncoding=UTF-8&serverTimezone=UTC
  jdbc.username = root
  jdbc.password = 123456
  ```

* 项目部署

  * ROOT.war
